# Portfolio ReactJS

Đây là dự án **portfolio cá nhân** được xây dựng bằng **React** và **Vite**, nhằm mục đích giới thiệu bản thân, các kỹ năng, dự án đã thực hiện và thông tin liên hệ. Giao diện được thiết kế **hiện đại, responsive**, hỗ trợ cả **desktop và mobile**.

---

## 🚀 Công nghệ sử dụng
-ReactJS, Vite,ReactRouter, Redux, ReactQuery, AntDesign,....

---
